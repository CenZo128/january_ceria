Git adalah version control -> sytem untuk track files.
Github adalah platform version control.

Command github :
1. Untuk menambah file -> git add <nama_file>
2. Untuk menaikkan file -> git commit -m "message"
3. Untuk mengirim file -> git push origin main

CASE :
Buatlah rumus luas segitiga.
Luas = 1/2 * alas * tinggi

ALGORITMA
1. Buat variable alas
2. Buat variable tinggi
3. Buat variable luas
4. Kalikan alas dan tingi dan 1/2
5. Pindahkan hasil perkalian ke variable Luas
6. Tampilkan variable luas

PSEUDOCODE

SET alas
SET tinggi
SET luas
STORE exact WITH 1/2

luas = alas * tinggi * exact

PRINT luas
